# Weather-Scraper

Scrapes www.foreca.fi for forecast in Finland.

## Installation

1. Download this project as zip and extract it
2. install BeautifulSoup and requests.
```bash
pip install beautifulsoup4
```
```bash
pip install requests
```
3. Run the application.
